# Governança e Documentação — Prompt para Agente Autônomo de Desenvolvimento

## Tarefa
Atualize o README do projeto e o arquivo de instruções do Copilot, garantindo a integridade e sincronização de todos os documentos do projeto.

## Requisitos

1. **Revisão e Consistência:**  
   Revise e mantenha a consistência entre os seguintes documentos:
   - `docs\agente_autonomo\AGENTE_AUTONOMO_ARQUITETURA.md`
   - `docs\agente_autonomo\AGENTE_AUTONOMO_BACKLOG.md`
   - `docs\agente_autonomo\AGENTE_AUTONOMO_CHANGELOG.md`
   - `docs\agente_autonomo\AGENTE_AUTONOMO_FAQ_LICOES_APRENDIDAS.md`
   - `docs\agente_autonomo\AGENTE_AUTONOMO_FEATURES.md`
   - `docs\agente_autonomo\AGENTE_AUTONOMO_HISTORIAS.md`
   - `docs\agente_autonomo\AGENTE_AUTONOMO_RELEASE.md`
   - `docs\agente_autonomo\AGENTE_AUTONOMO_RL.md`
   - `docs\agente_autonomo\AGENTE_AUTONOMO_ROADMAP.md`
   - `docs\agente_autonomo\AGENTE_AUTONOMO_TRACKER.md`
   - `docs\agente_autonomo\AUTOTRADER_MATRIX.md`
   - `README.md`
   - `.github\instructions\instrucoes.instructions.md`
   - `.github\copilot-instructions.md`

2. **Mecanismo de Sincronização Obrigatória:**  
   Sempre que qualquer documento citado acima for alterado, **obrigue** o Agente Autônomo a revisar e atualizar todos os outros documentos impactados para manter a integridade e sincronização.

3. **Mecanismos Explícitos de Controle:**  
   Implemente mecanismos explícitos (checklists automáticos, comentários, notificações ou validações automáticas) nos documentos para registrar e rastrear atualizações, garantindo transparência e documentação consolidada.

4. **Processamento Automatizado e Documentado:**  
   Processe mudanças de forma automatizada e documentada, sem exceção, em cada ciclo de alteração.

---

### Exemplo de mecanismo

Ao editar um documento, adicione um checklist automático que indique a revisão dos demais documentos. Caso uma revisão não seja realizada, solicite ou bloqueie o commit até que todos estejam atualizados.
